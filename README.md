# Stone Deco — Гостевой микросайт

Секретная страница для гостей открытия шоурума Stone Deco, Петрозаводск, 17 июня 2026.

## Как пользоваться

Гость получает физическую карточку с QR-кодом → сканирует → попадает на этот сайт.

## Структура страницы

- **Кто мы** — короткое описание компании
- **Материалы и изделия** — 5 плиток с ссылками на Яндекс.Диск
- **Сюрприз** — коллекция обоев на смартфон
- **Что вы получаете** — развилка по аудитории (4 варианта), после выбора появляется карточка со спецпредложением и кнопками
- **Маркетплейсы** — Озон и Яндекс Маркет
- **Контакты** — адрес, телефон, соцсети

## Как обновить ссылки

Все ссылки собраны в одном месте — в начале скрипта в файле `index.html`:

```js
const LINKS = {
  textures:      "https://...",   // Каталог текстур
  renders:       "https://...",   // Текстуры для проектов
  custom:        "https://...",   // Изделия на заказ
  decor:         "https://...",   // Декор из камня
  wallpapers:    "https://...",   // Обои на смартфон
  useful:        "https://...",   // Полезные материалы
  designers:     "https://...",   // Спецпредложение: дизайнеры
  builders:      "https://...",   // Спецпредложение: строители
  furniture:     "https://...",   // Спецпредложение: мебельщики
  ozon:          "https://...",   // Озон
  yandex_market: "https://...",   // Яндекс Маркет
  phone:         "tel:+7...",     // Телефон менеджера
  tg_manager:    "https://t.me/...", // Telegram
}
```

Найди нужный ключ, замени URL — остальное трогать не нужно.

## Деплой

Сайт работает на GitHub Pages. После загрузки `index.html` в репозиторий страница доступна по адресу:

```
https://[логин].github.io/[репозиторий]/
```
